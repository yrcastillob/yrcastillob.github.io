Employment Exchange - Bolsa de Empleo

Aplicación Java que simula una bolsa de empleo como proyecto final de una materia universitaria. Entre las principales funciones se encuentran la posibilidad de agregar hojas de vida, mostrar información a partir de la cédula de ciudadanía y eliminar participantes. El proyecto emplea una base de datos con ORM.
